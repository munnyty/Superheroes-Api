# Superheroes-Api
Описание функции:
В данном проекте была создана и протестирована функция по нахождению самого высокого супергероя из API по параметрам пола и наличия работы(булево значение)
Инструкция:
Для запуска тестов нужно перейти в директорию проекта(Superheroes-Api\Superheroes) с помощью git bash, либо cmd. Затем можно ввести команду "make help" после чего появятся всевозможные команды. Само тестирование запускается командой make test.
Если нет возможности работать с файлом make, можно в терминале python в нашей директории установить зависимости с помощью команды python -m pip install -r requirements.txt, а затем воспользоваться командой python -m pytest для теста без подробностей, либо python -m pytest -v -s для вызова тестирования с подробностями.
Имеющиеся команды (Если используете Makefile):
	make help - показывает полный список команд, помимо -help
	make install - Установка зависимостей 
	make test - Запуск всех тестов
	make test-detailed - Тесты с подробным выводом
